System Architecture

Unity6 Runtime

圧倒的な描画パフォーマンスを誇るUnityをレンダリングエンジンとして採用。多彩な画像表現力を持ちます。将来的には3Dモデルの表示や最新の画像表現まで幅広く対応可能なポテンシャルを秘めています。

WPF (.NET) Editor

　シンプルなコマンド体系。修正してクリックで再開したいところから即再生。複雑な操作を覚える必要はなく、ストレスフリーな編集・管理が可能です。

💻

ノーコード・ローコード開発

文章感覚で書けるVNScriptと、即座に結果を確認できるリアルタイムプレビューを搭載。

🎨

高度なレイヤー演出
